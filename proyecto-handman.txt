Juego del ahorcardo(Hangman)
LISTO---1-se necesita una base de datos de las posibles palabras a jugar.
LISTO---2-se necesita que el muñeco se genere segun el jugador falle una letra.
LISTO---3-al presionar el botón de una letra esta quedara deshabilitada.
LISTO---4-Si se ha completado la palabra, se mostrara un mensaje y se rinicia el jeugo apra pasar al siguiente nivel.
Descartado---5-tomar por igual las mayusculas y minusculas.
Descartado---6-validar que solo se ingresen letras.
LISTO---7-tener funciones separa para cada cosa (validaciones, base de datos)
DESCARTADO----8- se peude tener un menu donde el usuario escoja el area sobre la cual se debe adivinar la palabra (aniamles,comidas,objetos,naturazleza, medios de transporte etc...)
Descartado---9-puede correr todo en un ciclo while que se ejecutara segun la longtitud de la palabra al llegar a 0 game over.
LISTO---10- se debe comprabar que la letra que el usuario ingrese este en la palabra a descubrir, ademas de saber si la letra ya fue usada o no, tambien la palbra tiene por ejemplo varias letras repetidas por ejemplo (caracas) si el usuario digita la a se descubrirarn las 3 a presentes (_a_a_a_)
LISTO---11- en la base de datos no incluir palabras con acentos o caracteres especiales.
LISTO---12- Si se ha completado el dibujo del ahorcado, el juego termina y Jugador pierde.
LISTO---13- se cambio el modo de jeugo usando TKinter para hacerlo visual y mas interactivo.
